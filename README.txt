## Project Name

RAG Assistant with Ollama

## Overview

This is a GUI based Retrieval Augmented assistant.
It prompts the user to select txt documents to feed into the FAISS Vector Store and 
ask questions while the system reply basing on the fed documents content.

The RAG system help users retrieve specific information from documents.

## Installation

To start the rag system, install ollama
Take note of the project structure in your setup and run the main.py file
Wait for intallation of requirements
And ==check the usage walkthrough
